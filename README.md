# Bun Phaser Template

## Includes
* Phaser 3

## Setup

Get [Bun](https://bun.sh).

```
bun install
```

## Run

```bash
bun run build
bun run serve
```

Defaults to http://localhost:3000.

## Files

* `dist/index.html`: Main static entry page.
* `src/js/app.js`: Main app script.
* `src/css/main.css`: Custom CSS.
